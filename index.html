Castlist.

Capstone project. Web app to streamline theatrical audition process. As a director, you can create audition days with automatically generated time slots. An actor can sign up for these time slots and upload their headshot (the headshot is then sent to an AWS S3 bucket) as part of this process. On the day of the audition, the director has access to a Tinder style swipe function (that displays the headshots) to sort the actors according to their audition as well as add notes to their audition: this effectively eliminates the need for physical headshots (saving paper and money). The director is then able to view all of the auditionees by how they were sorted (or swiped), which allows for quick and easy callback list generation.
